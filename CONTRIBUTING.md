# Contributing to TapConnect

First off, thank you for considering contributing to **TapConnect**! 🎉

It's people like you that make open-source projects great.

## Getting Started

To set up your local development environment, please refer to the **Getting Started** section in the `README.md` file for detailed instructions on:

* Installing dependencies
* Configuring environment variables
* Setting up the Supabase database

## Prerequisites

Before contributing, ensure you have:

* **Node.js** installed on your local machine
* A **Supabase** account with a project configured

## How to Contribute

### 1. Create a Branch

Create a new branch for your feature, enhancement, or bug fix.

```bash
git checkout -b feature/your-feature-name
```

### 2. Make Your Changes

Implement your changes while following the project's coding conventions and architecture.

Please ensure your code aligns with the technologies used in the project:

* **Zustand** for state management
* **React Router** for navigation
* **Supabase** for backend services

### 3. Commit Your Changes

Write clear and descriptive commit messages.

```bash
git commit -m "Add new feature: description of feature"
```

### 4. Push Your Branch

Push your changes to your forked repository.

```bash
git push origin feature/your-feature-name
```

### 5. Open a Pull Request

Submit a Pull Request (PR) against the `main` branch of the original repository.

When creating your PR, please include:

* A clear description of the changes made
* The problem being solved
* Any relevant screenshots or demonstrations (if applicable)

## Code Style Guidelines

* Follow the existing project structure and coding patterns.
* Keep components modular and reusable.
* Use meaningful variable and function names.
* Ensure code is properly formatted before submission.

## Reporting Issues

If you find a bug or have a feature request, please open an issue and provide:

* A clear title and description
* Steps to reproduce (for bugs)
* Expected and actual behavior
* Screenshots or logs when applicable

## Code of Conduct

Please note that this project is released with a Contributor Code of Conduct.

By participating in this project, you agree to abide by its terms and help maintain a welcoming and respectful environment for everyone.

Be respectful, constructive, and collaborative in all discussions, issues, and pull requests.

---

Thank you for contributing to **TapConnect**! 🚀
